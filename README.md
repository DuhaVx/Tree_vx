# Binary Search Tree (BST)

Простая реализация бинарного дерева поиска на Python.

## Требования
- Python 3.8+ (допустимо выше)
- Опционально: менеджер `uv`, если хотите запускать через `uv run`

## Как запустить демонстрацию
Из корня репозитория выполните одну из команд:
- Стандартный Python: `python bst.py`
- Через uv (если установлен): `uv run python bst.py`

Скрипт создаёт дерево, вставляет несколько элементов, выводит высоту, проверяет балансировку, ищет и удаляет узлы. Результаты печатаются в консоль.

## Как проверить функции вручную
Откройте REPL и импорируйте класс:
- Старт Python: `python`
- Внутри REPL:
  ```python
  from bst import BinarySearchTree
  tree = BinarySearchTree()
  tree.insert(10, "root")
  tree.insert(5, "left")
  tree.search(5)           # вернёт "left"
  tree.delete(10)          # удаление корня
  tree.height()            # высота дерева
  tree.is_balanced()       # проверка баланса
  ```

## Тесты
Автоматических тестов нет. Для быстрой проверки достаточно запустить `python bst.py` и убедиться, что вывод соответствует ожидаемому сценарию.
